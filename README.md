# 🧠 AI-Based Mental Health Monitoring System

An AI-powered web application that predicts whether a person is likely to seek mental health treatment based on survey responses. The project uses **Machine Learning (Random Forest Classifier)** and is deployed with **Streamlit**.

---

## 📌 Project Overview

Mental health is an important aspect of overall well-being. Many people experience stress, anxiety, and other mental health challenges but may not recognize when professional support is needed.

This project uses a **Random Forest Classifier** trained on a mental health survey dataset to predict whether a person is likely to require mental health treatment based on their responses.

The application provides a simple and interactive interface where users can enter their information and receive an AI-based prediction.

---

## 🎯 Objectives

- Predict whether a person is likely to seek mental health treatment.
- Build an easy-to-use web application using Streamlit.
- Compare different machine learning models and select the best-performing model.
- Demonstrate an end-to-end machine learning workflow.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/mental-health-monitoring.git
```

Move into the project folder:

```bash
cd mental-health-monitoring
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
